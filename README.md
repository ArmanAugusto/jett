# Jett

## Usage

### Add a high-priority task
jett add "Upgrade server" --due 2025-03-04 --priority high

### Edit task #3 title and due date
jett edit 3 --title "Upgrade database server" --due 2025-03-10

### Delete task #2
jett delete 2

### List everything
jett list

### Show summary
jett summary
